▼ - Raccourcis clavier

Pour ne pas se faire choper, il faut connaître ces raccourcis clavier pour être beaucoup plus tranquille

Ctrl + w

Ferme instantanément la fenêtre
( - ATTENTION - certain jeux peuvent vous mettre une pop-up pour confirmer que vous perderez votre sauvegarde - ATTENTION - )

Ctrl + maj + T

Rouvre les derniers sites que vous avez fermé
( - ATTENTION - Les jeux qui demandait une confirmation avant de se fermer ne pourront pas être réouvert avec la sauvegarde - ATTENTION - )

Ctrl++D

Vous permet de créer un nouveau bureau et qu'aucune application ne parait encore lancé
( contrairement au Racourcits d'avant vous n'avez aucune chance de perdre vos données )

Ctrl++
→ ou ←

Vous permet de changer de bureau super rapidement et de ne pas être suspect

Échap

Sur ce site si vous appuyez une fois sur Échap vous êtes emmener sur la recherche Wikipédia (Attention si vous maintener Échap il ne se passera rien)
